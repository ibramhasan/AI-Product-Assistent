# MVP Scope

## Input

- Paste Product Backlog Item
- Paste User Story
- Paste Feature Description

## Output

- Customer Problem
- Customer Value
- Hypothesis
- Recommended Vertical Slice
- Why this slice?
- Recommended SPIDR Pattern
- INVEST Review
- What NOT to build yet
- Questions for Product Owner

## Out of Scope

- Jira Integration
- Confluence Integration
- PRD Upload
- Sprint Planning
- Capacity Planning
- Story Point Estimation
