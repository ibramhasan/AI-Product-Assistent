# Output Format

## Purpose

Menggabungkan seluruh hasil pipeline menjadi satu laporan yang konsisten dan mudah dipahami.

Laporan harus menunjukkan bagaimana AI mencapai setiap kesimpulan melalui evidence, reasoning, dan assumptions.

---

## Input

Seluruh hasil pipeline sebelumnya.

---

## Process

1. Susun hasil sesuai urutan pipeline.
2. Hindari duplikasi informasi.
3. Kelompokkan hasil berdasarkan tujuan analisis.
4. Pastikan seluruh rekomendasi memiliki evidence dan reasoning.
5. Tampilkan seluruh assumptions secara eksplisit.
