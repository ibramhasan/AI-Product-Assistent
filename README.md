An AI-powered assistant that helps Product Owners identify customer value, formulate hypotheses, recommend valuable vertical slices, and improve backlog quality using Agile principles.

## Principles

- Customer Value First
- Vertical Slicing
- Hypothesis Driven Development
- SPIDR
- INVEST
- Show Me The Software
